# CrimeGraph

## 4D Forensic Reconstruction using Scene Graphs

CrimeGraph is an advanced AI-powered system designed to revolutionize forensic investigations by transforming fragmented evidence into a cohesive, interactive 4D digital twin. By fusing textual reports, static imagery, and video surveillance footage, the system reconstructs a searchable, temporally accurate 3D model of a crime scene. This allows investigators to visualize events, query spatial relationships, and analyze the timeline of a crime with unprecedented precision.
